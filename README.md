# ❤️ Heart Disease Risk Prediction

An end-to-end Machine Learning project that predicts the likelihood of heart disease using clinical data, with an interactive web application for real-time inference.

---

## 🌐 Live Application

🔗 https://sushmitha109-heart-disease-prediction-app-wnjpaa.streamlit.app/

---

## 📌 Problem Statement

Heart disease is one of the leading causes of mortality worldwide. Early prediction using clinical parameters can significantly assist in timely diagnosis and treatment.

This project aims to build a predictive model that classifies whether a patient is at risk of heart disease based on medical attributes.

---

## 🧪 Project Demonstration

### 🔍 Data Analysis

* Explored dataset distributions and feature relationships
* Identified patterns using correlation analysis
* Detected outliers and inconsistencies

### 🧹 Data Preprocessing

* Handled missing values and data cleaning
* Applied feature scaling using StandardScaler
* Ensured clean and structured dataset for modeling

### ⚙️ Feature Engineering

* Selected relevant clinical features
* Maintained consistency between training and inference data
* Structured input pipeline for deployment

### 🤖 Model Development

* Trained using Gradient Boosting Classifier
* Evaluated using classification metrics
* Optimized for reliable prediction performance

### 🚀 Deployment

* Built interactive UI using Streamlit
* Integrated trained model and scaler using pickle
* Enabled real-time predictions with probability output
* Deployed on cloud with GitHub integration
